Twitcord

Twitcord is a web-based application that seamlessly combines the best features of Discord and Twitter. Built using Django as a web framework, Twitcord leverages a powerful tech stack comprising HTML, CSS, JS, and SQLite, ensuring a smooth and engaging user experience. With Twitcord, users can explore a vast array of topics tailored to their interests. Discover a new realm of social media with Twitcord, where the fusion of Discord and Twitter opens up a world of possibilities for connection, collaboration, and creativity!

Match Mania

Match Mania is a Java-based card matching game specifically designed for Android devices. Its objective is to match all the cards with the fewest number of attempts possible. The game was developed as part of my application development course, and it incorporates Firebase for storing and retrieving user scores, as well as implementing a convenient login system for players.
